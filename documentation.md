# DictaTranscribe — Project Bible (documentation.md) 

## Цель 
Android-приложение: диктофон + отправка записи на backend для транскрипции (Whisper) + экспорт текста. 

## Выбранная архитектура 
**Вариант B (Hybrid):** Android пишет/хранит аудио локально, backend (FastAPI в Docker) делает транскрипцию. 

## MVP-скоуп (фиксируем) 
- 1 пользователь, без аккаунтов 
- Запись аудио на Android 
- Отправка выбранной записи на backend → получение текста 
- Сохранение истории локально (Room) + просмотр текста 
- Share/Export TXT (аудио можно шарить как файл) 

## Безопасность (обязательные правила) 
- Не доверяем клиенту: проверки размера/типа/таймаутов — на backend 
- Секреты/ключи только в `.env` (и `.env` в `.gitignore`) 
- Backend не хранит пользовательские данные по умолчанию (stateless), если не решим иначе
